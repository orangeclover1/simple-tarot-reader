# Lantern Tarot v12 Mobile Navigation Fix

This build moves the main navigation away from the Android gesture/home-button area.

## Changes

- Main nav moved from the bottom of the app to the top.
- Nav height reduced from 68dp to 56dp.
- Button labels are slightly smaller so five nav items fit more cleanly.
- `Insights` is shortened to `Stats` in the top bar.
- Scroll pages include a 28dp bottom spacer so final buttons and text fields remain clickable above Android system navigation.
